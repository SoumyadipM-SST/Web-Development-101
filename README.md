## Activity

The **Activity** folder contains small practice exercises used to learn fundamental web development concepts.

### Topics Covered

* Basic HTML structure and semantic tags
* Lists, tables, and hyperlinks
* Forms and user input fields
* Images and media embedding
* CSS styling and layout
* Basic responsive design
* CSS shapes and transformations
* CSS animations

## ETP – Portfolio

The **End Term Project (ETP)** is a fully designed **personal portfolio website** created using HTML and CSS.

### Features

* Responsive design using **media queries**
* Clean navigation bar with smooth scrolling
* Structured sections including:

  * About
  * Skills
  * Projects
  * Work Experience
  * Contact
* Contact form UI
* Social links to email, LinkedIn, and GitHub
* Resume download option
* CSS variables for maintainable styling
* Hover effects and interactive UI elements

### Technologies Used

* HTML5
* CSS3
* Flexbox
* CSS Grid
* CSS Animations
* Media Queries for responsive design
